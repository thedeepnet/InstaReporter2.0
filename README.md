## Installation :

* `apt update -y && apt upgrade -y`
* `pkg install -y git`
* `pkg install python`
* `apt install git curl php wget -y`
* `git clone https://github.com/muneebwanee/InstaReporter2.0.git`
* `cd InstaReporter2.0`
* `python3 -m pip install requests`
#### > Run : `python3 InstaReporter2.0.py`

## Single Command :
```
apt update -y && apt upgrade -y ; pkg install -y git ; pkg install python ; apt install git curl wget php -y ; git clone https://github.com/muneebwanee/InstaReporter2.0.git ; cd InstaReporter2.0 ; python3 -m pip install requests ; python3 InstaReporter2.0.py
```
